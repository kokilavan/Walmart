## Walmart Sales & Profit Dashboard (Power BI)
Built an interactive Walmart Sales Dashboard using Power BI to analyze sales, profit, quantity, and profit margin across categories, regions, and time. The report features KPIs, slicers, trend analysis, and drill-throughs to enable detailed performance and business insights.

### Business Objective
Analyze Walmart sales, profit, and quantity performance across categories, regions, and time.

### Tools Used
- Power BI
- SQL
- Excel

### Key Features
- KPI cards for Sales, Profit, Quantity, Profit Margin
- Category, Region, and Segment analysis
- Monthly sales trend
- Drillthrough to order-level details

### File Structure
- walmart.sql : SQL Scripts for analyzing Walmart sales, profit, and quantity performance across categories, regions, and time.
- walmart.pbix : Power BI dashboard that visualizes the SQL results and provides a user-friendly interface for exploring the data.

### Outcome
Enabled management to identify high-performing categories and analyze transaction-level data.
